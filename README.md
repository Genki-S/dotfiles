# Dotfiles

My setting files.

## Installation

### Prerequisits
* the\_silver\_searcher
* hub

### Install vendor applications

```
vim -u ~/dotfiles/vendor/neovendor.vim
```

Installs in `~/vendor`.

### Install Task

```
rake install
```

I hope this works.
